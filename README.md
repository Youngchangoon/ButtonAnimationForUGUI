
# ButtonAnimationForUGUI
Plugin for easy way to add button animation in UGUI

---

UGUI 버튼을 이용해 커스터마이징한 버튼들입니다.

## Description

---

유니티에서 지원하는 기본 버튼들은 눌렀을때 색상변화만 지원합니다.  
그리고, 보기에 썩 아름답지도 않죠.  
이 플러그인은 버튼을 눌렀을때 반응하는 버튼들을 쉽게 만들 수 있습니다!


## Requirement

---

DoTween ( + v1.2.235 )
